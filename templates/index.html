Step 1 — Classify
Naive Bayes reads your question and predicts SQL intent (AGGREGATE, JOIN, FILTER…) with confidence score.

Random Forest uses 17 features to estimate query success probability before the LLM call is made.
Step 3 — Generate
Intent hint injected into prompt. LLM generates dialect-aware SQL for SQLite, MySQL, or PostgreSQL.
